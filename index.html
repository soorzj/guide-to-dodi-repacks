‚û• How to Install Games

If you have done a DDL, download 7-ZIP, select all parts, and use 7Z to extract them. (On Windows 11, you'll have to click "More Options" to find 7Z.) Once they're done, open the folder and run the setup. It is not recommended to delete the parts until the game is verified to work.

If you want to play a game online, you may have to go to online-fix.me for the online fix.

If you did the torrent, simply run the setup.

‚û• Fixing Common Errors

It is common to encounter various errors and problems during/after installation of DODI Repacks. Here are some common fixes:

  • Install to games, NOT Program Files (x86).
  • Turn off antivirus during setup.
  • Add an exclusion to the directory where you normally install your games (e.g., C:\Games).
  • Un-select the "DirectX 9.0c" option unless it's your first time.
  • Prefer installing onto a shorter path.

This is a small list of common fixes, and your issue might not be solved by these steps.
